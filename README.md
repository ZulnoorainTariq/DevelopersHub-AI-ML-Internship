# DevelopersHub AI/ML Internship Tasks

## Task 1: Iris Dataset Exploration & Visualization

### Objective
Explore and visualize the Iris dataset to understand feature distributions, relationships, and outliers.

### Dataset
- Name: Iris Dataset
- Source: Seaborn built-in
- Size: 150 rows × 5 columns

### Steps Performed
1. Loaded and inspected the dataset
2. Checked for missing values
3. Scatter plots
4. Histograms
5. Box plots
6. Correlation heatmap

### Key Findings
- No missing values in the dataset
- Setosa is clearly separable from other species
- Petal Length & Width are the strongest features
- Petal Length and Width have very high correlation (0.96)

### Libraries Used
pandas, seaborn, matplotlib
